Posts to www.AxisAndAllies.org

This poster is build for PBF games via AxisAndAllies.org. Instructions:

  1. Create a new Forum post in the Play Boardgames section
  2. Copy the topic id from the URL displayed in the address bar of you browser. If the URL is https://www.axisandallies.org/forums/topic/32423/triplea-test-posts, put this 5 digit number (32423) into the Topic ID field in TripleA
  3. Put your login and password for the AxisAndAllies.org forum into the login and password fields
  4. Click the Test Post button, to check that TripleA can reply to your forum post

Note: Your AxisAndAllies.org login and password are not stored as part of the save game, but they are stored encrypted in the local file system if you select the option to remember your credentials. You may have to enter your login and password again if you open the save game on another computer.

* Make sure you have registered for receiving dice emails if your dice server sends them out. Click the 'Help?' button by your dice server to see if this is required.
