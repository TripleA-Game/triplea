Posts to forums.triplea-game.org

You can play PBEM/PBF games via forums.triplea-game.org.

Instructions:

  1. Create a new Forum post in the Play by Forum category https://forums.triplea-game.org/category/6/
  2. Copy the topic id from the URL displayed in the address bar of you browser. If the URL is https://forums.triplea-game.org/topic/24/ put the topic number (24) into the Topic ID field in TripleA
  3. Put your login and password for the forums.triplea-game.org forum into the login and password fields
  4. Click the Test Post button, to check that TripleA can reply to your forum post
* Make sure you have registered for receiving dice emails if your dice server sends them out. Click the 'Help?' button by your dice server to see if this is required.
You can register at dice.tripleawarclub.org by entering your email address!
